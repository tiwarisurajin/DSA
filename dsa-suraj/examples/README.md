# Examples

Tiny, visual, educational scripts — not interview problems. These exist to build intuition for how things work under the hood (memory layout, recursion call stacks, hashing), separate from `implementations/` which is organized around solving problems.

| Example | Demonstrates | Status |
|---|---|:---:|
| [`recursion_call_stack.py`](./recursion_call_stack.py) | Visualizing stack frames during recursion | ✅ |
| Hash table collision visualization | How hashing + collision resolution actually works | ⬜ queued |
| Heap array-vs-tree visualization | Why a heap is stored as a flat array | ⬜ queued |
| Graph traversal step-by-step printer | DFS/BFS order side by side on the same graph | ⬜ queued |
